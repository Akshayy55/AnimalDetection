# Scripts to run Animal Detection and store Data into the DB and also Retrive back from the DB.

## step1
```
Run ```main.py file```
and run on local host the get API's
```

## step2
To Insert Data Into the DB 
use  this API 
```/entry_animal_info```

## step3
To Retrive Animal Info Data from the DB
use this API
```/get_animal_info```
